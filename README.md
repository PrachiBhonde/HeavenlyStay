HeavenlyStay

HeavenlyStay is a full-stack hotel booking web application designed to offer users a seamless and convenient experience for discovering and booking hotels online.  
It provides a responsive interface, secure authentication, and smooth booking functionality with image uploads managed via Cloudinary.

Tech Stack

1.Frontend
- HTML  
- CSS  
- JavaScript  
- Bootstrap  

2.Backend
- Node.js  
- Express.js  
- MongoDB  
- Passport.js (for Authentication)  
- Cloudinary (for Image Storage)


Features
- User Authentication — Secure login and signup using Passport.js  
- Room Booking System — Users can browse and book rooms easily  
- Image Uploads — Cloudinary integration for hotel/room images  
- Responsive Design — Optimized for all screen sizes using Bootstrap  
- Database Integration** — MongoDB for dynamic data storage
